CHATAPP

OVERVIEW

Welcome to Chat App! Users can send images, take photos, send their location, and chat with others. Using Firebase to store messages, and images, while using Dialogflow for the in-app chatbot. Yes you can talk with a chat-bot! Simple and understandable, using Gifted Chat to help design and setup the UI component, it will feel natural to use.

OBJECTIVE

To build a chat app for mobile devices using React Native. The app will provide users with a chat interface and options to share images and their location.
Extra: Integrate a chat bot to help respond to users questions about chat app.

TOOLS

React Native, Expo, Google Firebase, Google Dialogflow

CLIENT-SIDE

First I need to set up a development environment to work with, React Native and Expo were used. Designing the app's UI '& Accessibility functionality with Native UI components and chat features from the Gifted Chat library.

LOGIN DESIGN

The app must be styled according to the given screen design. Testing my skill to replicate a set style for the Chat app finished product.

CHAT UI '& ACCESSIBILITY

Building the chat screen UI used many key points from popular chat app interfaces. Better to use designs that work and are natural to many users. Next was the chat functionality, using the Gifted Chat library, which made it very streamlined to connect and set message-sending.

STORING DATA ON THE CLIENT-SIDE

What if you lost your internet connection and needed to view an old message sent? Phase 4 added the feature to have messages stored locally using async Storage. This allowed users to view older messages, once online you could send messages and receive updated messages.

BACKEND

Here I set up Real-Time Applications '& Data Storage, which includes Authenticating users anonymously with Firebase and storing conversations in the Firestore Database.

FIREBASE

Firebase provided by Google helps host databases, services, authentication, and integration for a variety of applications. This was perfect for the Chat app. Using their authentication users can anonymously sign in with Firebase. With Firebase: Firestore helps store conversations, photos, and videos in their database. Very helpful when need to read an old message sent by a friend or yourself.

DIALOGFLOW

Dialogflow provided by Google is a natural language understanding platform used to design and integrate a conversational user interface into mobile apps. With Dialogflow integrated into the Chat App, I was able to design a bot to respond to questions asked by users. From App related questions to very basic conversations ( eg. Hello, Can I ask you a question, What is a Chat App, )

COMMUNICATION FEATURES

Providing users with the ability to send Images, and take photos from their device library or device camera. Using Google Firebase Cloud Storage to store images and send messages via Gifted chat. Making use of Reacts Native map to allow users to send current location in a map view via Gifted chat.

APPROACH

CHALLENGES

When looking back integrating the chatbot was the most challenging and fun experience. Since this was the first time I added an extra feature that was not in the main objective. Finding ways to get it to work with gifted chat, learning about hiding secret keys through environment variables. I had to search through multiple documents, took lots of trial and error but in the end, I learned so much from it.

DURATION

Development of Chat App had its ups and down with 2 weeks for its completion.

LOOKING BACK

The Chat app's objective is to provide users with a chat interface and options to share images and their location. Furthermore adding a chatbot feature so users can be informed of features and descriptions within the application.

Next time I would like to tackle multiple chats, this means users can have saved private messages and public group messages. It would help complete the goal of a Chat app, other smaller changes would be adding resizing of fonts, and image backgrounds besides set colors.

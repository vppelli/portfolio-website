About me

As a former 3D Modeler turned web developer, I create prototypes, render-ready assets, and high-quality textures for games and simulators. Using this knowledge, I've learned to develop website concepts into working prototypes, from simple Javascript apps to Movie listing web pages using API databases and authentication for creating accounts. Art has been my driving passion for trying new experiences, from creating early game UI prototypes to coding applications using HTML, CSS, and JavaScript. My creative touch has grown, improving user experience and clean code so anyone can read and understand my process. Some examples of my work include.

  • Pokedex: This app uses the PokeAPI to display information about selected Pokemon. It provides a list of Pokemon with information about them. It's developed using HTML, JavaScript, Bootstrap, and PokeAPI.
  • MovieMike's: Provides users with access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies. It's developed using React, JavaScript, Bootstrap, MovieAPI, and Netlify.
  • Mobile Chat App: This app provides users a chat interface and options to share images and their location. It can also chat with AI. It's developed using JavaScript, React Native, and Android Studio.

More About Me

I have a strong background in Art, having completed 2 years of courses in Architecture at Essex County College and received an Advanced 3D Modeling Diploma from CG Spectrum College of Digital Art and Animation. Worked as a 3D Environment Artist at SoulLes Games and Metabuff, where I created realistic and stylized environment assets, props to creatures, and full game environments. With multiple certifications in 3D modeling this helped build up many skills, being proficient in PBR textures, UV tiling, and optimizing game ready assets.

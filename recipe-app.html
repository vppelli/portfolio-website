RECIPEAPP

OVERVIEW

Welcome to the Recipe App! This web application provides information about different Recipes, ingredients, and difficulty. Users can sign up, upload their recipes, and view or update a list of recipes. I use PostgreSQL to store lists of Recipes, Ingredients, and Images. Simple and understandable, using Django on Python to provide a responsive application for all users.

OBJECTIVE

Using React, build the client-side for MovieMike based on its existing server-side code (Movie API and database). Integrate features that allow account creation, and access to film information such as genres, directors, and similar films. Give the ability to customize a list of favorite movies that they can add or remove anytime.

TOOLS

Python, Django, JavaScript, HTML, CSS

DESIGN

Many Recipe app-related websites are noisy and difficult to navigate. Users often struggle to find specific information about Recipes, and ingredients, including quality-of-life improvements such as difficulty recommendations.
I referenced the most popular websites. A home page that presents all needed navigations. Something that users are familiar with and can be improved upon freely.

With the Search function at the bottom of the screen, users will always have quick access on mobile. Desktop and computer viewing also use the same layout. Allowing scrolling through recipes and searching from the bottom when scrolling.

Users can click on an image of a recipe and be redirected to its enlarged page. I designed this area to present the recipe image on the background with a title and bio followed by a difficulty tag on the right, keeping the ingredients used at the bottom.

At the bottom under ingredients, you can interact with the edit button to fix or update that recipe.
